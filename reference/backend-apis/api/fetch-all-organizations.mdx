---
title: "Fetch all organizations"
api: "GET https://{auth_url}/api/backend/v1/org/query"
---

### Path Parameters

<ParamField path="auth_url" type="string" required>
  The domain for your application
</ParamField>

### Query Parameters

<ParamField query="page_size" default="10">
  The number of results to return at a time. Must be between 1 and 100.
</ParamField>

<ParamField query="page_number">Used to page over results</ParamField>

<ParamField query="order_by">
  How to order the results. Must be one of: `CREATED_AT_ASC`, `CREATED_AT_DESC`,
  `NAME`
</ParamField>

<ResponseExample>

```json
{
  "total_orgs": 21,
  "current_page": 0,
  "page_size": 10,
  "has_more_results": true,
  "orgs": [
    {
      "org_id": "d488996d-8ccc-4101-b5f2-131f5f09ddb6",
      "name": "OneOfYourCustomers"
    },
    {
      // There will be 10 orgs in this response
    }
  ]
}
```

</ResponseExample>
