---
title: "Fetch user by username"
api: "GET https://{auth_url}/api/backend/v1/user/username"
---

### Path Parameters

<ParamField path="auth_url" type="string" required>
  The domain for your application
</ParamField>

### Query Parameters

<ParamField query="username">The username to look up</ParamField>

<ParamField query="include_orgs" default="false">
  Whether or not to include the user's organization information in the response.
  Default false
</ParamField>

Successful response is the same as
[Fetch user by id](/reference/backend-integration/api/fetch-id).
