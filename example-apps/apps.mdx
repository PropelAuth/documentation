Listed below are a few guides to help get your started building applications using PropelAuth libraries. We also have [Github repos](https://github.com/PropelAuth) listed for all of these applications for you to clone and explore on your own, which have been separated into frontend and backend starter repos.

| [![Next Example App](/images/example-app-nextjs.png)](https://www.propelauth.com/post/nextjs-starter-application) <p>In this guide, we’ll build an example B2B application in Next.js where users can sign up, login, manage their accounts, and view organization and member information, all using PropelAuth and **NextJS**.</p> | [![React/Express Example App](/images/example-app-react-express.png)](https://www.propelauth.com/post/react-express-starter-app) <p>In this guide, we’ll build an example B2B application in React where users can sign up, login, manage their accounts, and view organization and member information, PropelAuth, **React**, and **Express**.</p>            |
| [![React/Fast API Example App](/images/example-app-react-fastapi.png)](https://www.propelauth.com/post/react-fast-api-starter-guide) <p>In this guide, we’ll build an example B2B application in React where users can sign up, login, manage their accounts, and view organization and member information, all using PropelAuth, **React**, and **FastAPI**.</p> | [![React/Flask Example App](/images/example-app-react-flask.png)](https://www.propelauth.com/post/react-flask-starter-app) <p> In this guide, we’ll build an example B2B application in React where users can sign up, login, manage their accounts, and view organization and member information, PropelAuth, **React**, and **Flask**.</p>|
